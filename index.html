SpiMoSim

About the library

A modular javascript library for interactive physical models (mostly agent based models) by Pascal Grafe

Where does it run?

If you see an animated SpiMoSim logo on top of this page, SpiMoSim is running right now in your browser! 1

Every application built with SpiMoSim will 2 run in every modern Browsers on desktop computers as well as on smartphones and tablet computers because it only relies on web technologies like HTML, Javascript and CSS. (Java is dead!)

How to build applications with SpiMoSim?

You do not need to be a web developer to get startet with SpiMoSim. You need to know a little Javascript. Download the example application (Ising model) and give it a try!
